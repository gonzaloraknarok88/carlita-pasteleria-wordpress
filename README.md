# Los Deleites de Carlita - WordPress

Sitio web de pastelería artesanal adaptado a WordPress desde diseño v0.

## Estructura del Sitio (Diseño v0)

### Secciones:
1. **Header** - Logo, navegación, botón Hacer Pedido
2. **Hero** - Carrusel animado, slogan, estadísticas
3. **Catálogo** - Grid de tortas con filtros
4. **Menú** - Lista de sabores disponibles
5. **Nosotros** - Historia de la pastelería
6. **Contacto** - Formulario + info de contacto
7. **Footer** - Enlaces, redes, copyright
8. **WhatsApp Button** - Botón flotante

## Implementación en WordPress

### Paso 1: Cambiar a tema de bloques (Hecho)
- Tema: Assembler (block theme)
- Acceder a: WordPress Admin > Apariencia > Temas

### Paso 2: Editar la plantilla de Inicio
- Ir a: Apariencia > Editor
- Seleccionar: Plantilla > Inicio del blog
- Reemplazar contenido con bloques de WordPress

### Paso 3: Colores del Tema
- Color primario: #E8A84D (dorado)
- Color secundario: #1A1A1A (negro)
- Fondo: #FDF8F3 (crema)
- Acento: #10B981 (verde WhatsApp)

### Paso 4: Tipografía
- Títulos: Playfair Display
- Cuerpo: Inter / System UI

### Paso 5: Componentes a construir con bloques
- Header: Bloque de navegación + logo
- Hero: Bloque Cover con carrusel de imágenes
- Catálogo: Grid de bloques de imagen + botón
- Contacto: Formulario de contacto + info
